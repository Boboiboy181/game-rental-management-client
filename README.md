## Cách chạy code:
- Pull code về máy
- Chạy lệnh bên dưới để cài đặt các dependency
```bash
$ yarn install
```
- Chạy xong, vào terminal gõ 1 trong các lệnh bên dưới
```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
